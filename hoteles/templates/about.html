Funcionamiento del servidor:

Todas las URL que se describen a continuación son accesibles tanto al escribirlas directamente en la barra del navegador como al pinchar en sus respectivos enlaces, que se pueden encontrar dentro de la página web

http://127.0.0.1:8000 >>> La página principal del servidor

Arriba, el banner del sitio

Arriba a la derecha el cuadro para loguearte, o tu nombre de usuario si ya estas logueado

Debajo del banner, barra con enlaces a diferentes páginas del servidor

Mitad derecha, lista con las selecciones de hoteles de los usuarios

Mitad izquierda, lista con los 10 hoteles más comentados

Abajo, pie de página con el "disclaimer"

http://127.0.0.1:8000/alojamientos >>> Lista con todos los alojamientos que contiene la base de datos del servidor

Arriba, el banner del sitio

Arriba a la derecha el cuadro para loguearte, o tu nombre de usuario si ya estas logueado

Debajo del banner, barra con enlaces a diferentes páginas del servidor

Debajo de la barra de enlaces, buscador de alojamientos en función de diversas características

Mitad derecha, lista con las selecciones de hoteles de los usuarios

Mitad izquierda, lista con todos los alojamientos, o con los que coincidan con las condiciones de busqueda

Abajo, pie de página con el "disclaimer"

http://127.0.0.1:8000/alojamientos/id >>> Información detallada sobre el alojamiento que corresponda con "id"

Arriba, el banner del sitio

Arriba a la derecha el cuadro para loguearte, o tu nombre de usuario si ya estas logueado

Debajo del banner, barra con enlaces a diferentes páginas del servidor

Mitad derecha, lista con las selecciones de hoteles de los usuarios

Mitad izquierda, lista con todos los alojamientos, o con los que coincidan con las condiciones de busqueda

Debajo de las listas, y solo si estas logueado, cuadro para añadir comentarios al alojamiento y botón para agregar el alojamiento a la selección personal del usuario

Abajo, pie de página con el "disclaimer"

Abajo, pie de página con el "disclaimer"
